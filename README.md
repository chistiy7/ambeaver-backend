## Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Docker](https://www.docker.com/)
- [Docker Compose](https://docs.docker.com/compose/)

## Running the app

```bash
# development
$ docker compose up

# production mode
$ docker compose -f docker-compose.prod.yaml up --build -d
```
